# Artecルービックキューブロボットサンプル

## 設計上の注意事項

アーム先部分の設計情報は同フォルダの```アーム先.step```にあります。

## ソフトウェア面の注意事項

ソルバー(解法探索部分)には高速化のためCythonを使用しています。これは標準では使えないので

```
pip3 install cython
```

のコマンドを実行しておく必要があります。

ソルバーは```solver.pyx```に書いてあるのですが、コンパイルは以下のようにします。

```
python3 setup_exe_cython.py build_ext --inplace
```

そして標準入力に

```
solver.pyx
```

と入力します。

## 基本的な使い方

Studthフォルダで下記を実行します。
```
python main.py
```
